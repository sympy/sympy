Overview
--------

The solvers module in SymPy provides various functions for solving different types of equations:

- Algebraic equations (using :func:`sympy.solvers.solvers.solve`)
- Set-based equations (using :func:`sympy.solvers.solveset.solveset`)
- Numerical equations (using :func:`sympy.solvers.solvers.nsolve`)
- Differential equations (using :func:`sympy.solvers.ode.dsolve`)
- Partial differential equations (using :func:`sympy.solvers.pde.pdsolve`)
- Linear systems (using :func:`sympy.solvers.solvers.solve_linear_system`)
- Polynomial systems (using :func:`sympy.solvers.polysys.solve_poly_system`)
- Inequalities (using :func:`sympy.solvers.inequalities.solve_univariate_inequality`)

Each solver is designed to handle specific types of equations and provides appropriate methods for finding solutions. The documentation for each solver includes detailed examples and usage instructions.

For more information on using these solvers, see the :doc:`/tutorials/intro-tutorial/solvers` tutorial.
